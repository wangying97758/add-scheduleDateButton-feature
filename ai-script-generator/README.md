# AI Table Script Generator Skill

这个 Skill 帮助 Claude 生成生产级别的 AI 表格（钉钉表格）脚本插件代码。

## 功能特性

- ✅ **严格遵循 API 规范**：只使用 `./references/api` 和 `../shared/references/api` 目录中定义的 API
- ✅ **生产就绪**：包含完整的错误处理和用户反馈
- ✅ **类型安全**：使用 `Base.isFieldOfType()` 进行类型检查
- ✅ **批量处理**：自动实现批量操作（每批 50-100 条记录）
- ✅ **完整输出**：生成 script.ts、manifest.json 和 description.md

## 使用方法

### 在 Claude.ai 中

1. 将 `.skills/ai-script-generator` 文件夹上传到 Claude.ai 的 Skills 设置中
2. 启用该 Skill
3. 自然语言描述你的需求，例如：

```
创建一个脚本，用于验证所有邮箱地址的有效性
```

```
生成一个脚本，可以批量替换文本字段中的内容
```

```
写一个脚本，可以随机填充数字字段的值
```

### 在 Claude Code 中

1. 将 `.skills/ai-script-generator` 文件夹复制到项目的 `skills/` 目录
2. Claude Code 会自动加载该 Skill
3. 直接在对话中提出需求

## 生成的输出

Skill 会根据你的需求生成以下内容：

### 1. TypeScript 脚本 (script.ts)
包含：
- 用户输入收集（使用 Input.formAsync）
- 数据验证逻辑
- 批量处理实现
- 错误处理和用户反馈

### 2. Manifest 文件 (manifest.json)
包含：
- 插件元数据
- 多语言描述
- 图标配置
- 脚本代码（已转义）

### 3. 描述文件 (description.md)
包含：
- 脚本名称
- 简短描述
- 详细介绍和使用说明

## 代码质量保证

生成的代码遵循以下原则：

1. **API 合规性**：只使用文档中定义的 API 方法
2. **错误处理**：所有操作都包裹在 try-catch 块中
3. **用户反馈**：使用 Output 方法提供进度反馈
4. **类型安全**：使用类型检查确保类型正确性
5. **批量处理**：自动实现批量操作以避免限制
6. **清晰注释**：关键逻辑添加注释说明

2. **类型匹配**：生成的代码会确保值类型与字段类型匹配
3. **批量限制**：自动处理批量操作的限制（50-100 条/批）
4. **错误处理**：所有生成的代码都包含完整的错误处理

## 示例对话

### 用户：创建一个脚本，可以验证邮箱地址

1. **script.ts** - 完整的邮箱验证脚本
2. **manifest.json** - 插件配置文件

生成的脚本会：

- 批量更新所有记录
- 提供详细的反馈信息

1. 一个完整的查找替换脚本
2. 支持正则表达式匹配
3. 显示替换预览
4. 让用户确认后再执行
5. 批量更新操作
