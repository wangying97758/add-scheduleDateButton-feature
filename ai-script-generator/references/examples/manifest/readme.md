# Manifest Examples

This directory contains example manifest.json files for reference when creating AI Table script plugins.

## Available Examples

- `checkdup.manifest.json` - Example for checking duplicate records
- `meta.manifest.json` - Example for extracting metadata
- `vote.manifest.json` - Example for voting functionality

## Usage

Refer to these examples to understand the correct structure and format for manifest.json files when using the ai-script-generator skill.
