### 字段类型 进度
### code
`progress`
### 读取
```typescript
interface Property {
  /** 进度格式 */
  formatter: ProgressFormatter;
  /** 是否开启自定义进度条值 */
  customizeRange: boolean;
  /** 起始值 */
  min: number;
  /** 目标值 */
  max: number;
}
```
### 写入
```typescript
interface Property {
  /** 进度格式，默认为PERCENT */
  formatter?: ProgressFormatter;
  /** 是否自定义进度条值，默认false */
  customizeRange: boolean;
  /** 起始值，默认0 */
  min: number;
  /** 目标值，默认1 */
  max: number;
}
```
### 备注
[ProgressFormatter](./API%20类型定义.md)

:::
当需要自定义起始值或目标值时，请置customizeRange为开启状态。
:::

## 字段值类型定义

### 字段类型 文本
### code
`text`
### 读取
`string`
### 写入
`string`

### 字段类型 数字
### code
`number`
### 读取
`string`
### 写入
`string | number`

### 字段类型 日期
### code
`date`
### 读取
`number`

以毫秒为单位的 Unix 时间戳
### 写入
`string | number`

dayjs可识别的时间文本结构或以毫秒为单位的 Unix 时间戳数值

### 字段类型 单选
### code
`singleSelect`
### 读取
```typescript
type Cell = {
  /** 选项ID */
  id: string;
  /** 选项名 */
  name: string
};
```
### 写入
```typescript
type Cell = string | {
  /** 选项ID */
  id?: string;
  /** 选项ID */
  name?: string;
};
```

直接传入选项名或选项ID或传入选项对象。
